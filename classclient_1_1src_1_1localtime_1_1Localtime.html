Detailed Description

The local time class provides convenience functions for the various animations needing to keep time.

The instance updates itself asychronously from the animation at the requested update rate. If the update rate is set to 0 the automatic update is disabled. This is useful when testing for a particular time.

See also
https://docs.python.org/3/library/datetime.html
