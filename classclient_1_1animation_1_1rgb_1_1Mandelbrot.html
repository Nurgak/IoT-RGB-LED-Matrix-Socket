Detailed Description

Fractal animation based on the Mandelbrot set.

The iteration is limited to 512 and the coloring is done using a 3-bit, 3D Hilbert curve, thus all the color values are displayed as 512 iterations is equal to (2**3)**3 = 512 colors.
