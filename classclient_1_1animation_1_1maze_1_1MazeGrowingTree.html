Detailed Description

Maze generator animation class.

MazeGrowingTree animation example

Recursive animation showing a maze being generated using the growing tree algorithm, inspired from https://weblog.jamisbuck.org/2011/1/27/maze-generation-growing-tree-algorithm. The forward pass is displayed in in green while the backwards pass is red. At the end of the animation the screen resets and starts over from a random position.
